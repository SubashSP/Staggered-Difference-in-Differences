# Staggered Difference-in-Differences
### Theory, Assumptions & Implementation — Callaway & Sant'Anna (2021)

> **Lecture notes and replication code** for the staggered DiD workshop, covering the Callaway & Sant'Anna (2021) estimator with full implementations in **Stata**, **R**, and **Python**.

---

## 📚 Overview

Standard Two-Way Fixed Effects (TWFE) DiD is biased when treatment effects are heterogeneous across groups and time — a problem that affects most modern policy evaluations with staggered rollouts. This repository provides:

- Full lecture notes on theory, assumptions, and implementation
- A self-contained dummy dataset (simulated minimum wage & employment panel)
- Parallel implementations in Stata, R, and Python
- Event study plots and aggregation strategies

## 📊 Dataset: `minwage_panel.csv`

A simulated balanced panel of **50 US states × 20 years (2000–2019)** modelling state minimum wage increases and their effect on log teen employment.

| Variable      | Type    | Description |
|---------------|---------|-------------|
| `state_id`    | int     | State identifier (1–50) |
| `year`        | int     | Calendar year (2000–2019) |
| `lemp`        | float   | Log teen employment (outcome) |
| `mw_treat`    | 0/1     | = 1 if state raised MW in or before this year |
| `treat_year`  | int/NA  | Year state first raised MW (NA = never treated) |
| `gvar`        | int     | Cohort: first treatment year; **0 = never treated** |
| `lpop`        | float   | Log state population (covariate) |
| `lincome`     | float   | Log median household income (covariate) |
| `ever_treated`| 0/1     | = 1 if state ever raises MW |

**Treatment cohorts:** States adopt MW increases in 2004, 2007, 2010, or 2013. ~25% of states are never treated.

## 🔢 Key Concepts Covered

| Topic | Description |
|-------|-------------|
| Classical DiD | 2×2 case, TWFE regression |
| Staggered treatment | Why TWFE fails with heterogeneous treatment effects |
| Goodman-Bacon decomposition | Quantifying TWFE contamination |
| ATT(g,t) | Group-time average treatment effects |
| Callaway & Sant'Anna | Two-step estimator: estimation + aggregation |
| Parallel trends | Conditional vs. unconditional; testing pre-trends |
| No anticipation | Assumption and validation |
| Aggregation | Simple, group, calendar-time, and event-study ATTs |
| Inference | Clustered SEs, wild bootstrap |
| Comparison | CS vs. TWFE vs. Sun-Abraham vs. Borusyak et al. |

## 🚀 Getting Started

### Stata

```stata
cd "path/to/staggered-did"
do code/staggered_did.do
```

**Required packages** (install once):
```stata
ssc install csdid
ssc install drdid
ssc install bacondecomp
ssc install coefplot
ssc install pretrends
ssc install eventstudyinteract
ssc install reghdfe
ssc install ftools
ssc install estout
```
> Requires **Stata 16+** for `csdid` (uses Stata frames).

---

### R

```r
# Install required packages (once)
install.packages(c("did", "tidyverse", "ggplot2", "fixest", "bacondecomp", "modelsummary"))

# Run the analysis
source("code/staggered_did.R")
```

Key package: [`did`](https://bcallaway11.github.io/did/) by Callaway & Sant'Anna.

---

### Python

```bash
pip install pydid pandas numpy matplotlib scipy statsmodels linearmodels
python code/staggered_did.py
```

Key package: [`pydid`](https://github.com/d2cml-ai/pydid) — Python port of the `did` R package.
